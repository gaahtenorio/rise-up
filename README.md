# Rise-up
Repositório criado para o desenvolvimento em grupo de uma aplicação para o programa de residência tecnológica do Porto Digital.

## Sobre o projeto
Criamos um site chamado Portal Disec, que é uma plataforma desenvolvida em Python utilizando Flask, destinada ao gerenciamento centralizado de informações de agências distribuídas nacionalmente.

A aplicação permite consultar dados operacionais, acompanhar indicadores de desempenho, monitorar eficiência energética, gerenciar documentos técnicos, controlar vistorias de segurança e administrar usuários com diferentes níveis de acesso.
